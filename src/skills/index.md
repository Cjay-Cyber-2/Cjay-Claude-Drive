# Skills Module

Skills are prompt-based commands that the model can invoke (20 files).

## Key Files

| File | Purpose |
|------|---------|
| `bundledSkills.ts` | Bundled skill registry |
| `loadSkillsDir.ts` | Load skills from directory |
| `mcpSkillBuilders.ts` | Build skills from MCP |
